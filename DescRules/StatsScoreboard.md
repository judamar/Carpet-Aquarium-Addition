# How to use the new scoreboard system:

## Usage:
`/sb <category>.<Item/mob id> <where to display>`

You can display the scoreboard in these locations:

```
list: Tab list
belowName: Below player name
sidebar: list on the side of the screen
```

### If you don't specify a location, it defaults to sidebar.
Categories:

```
b: Item broken
c: Crafted item
d: Dropped item
k: Mob killed
m: Block mined
p: Item picked up
u: Item used
killedBy: Killed by mob
```
